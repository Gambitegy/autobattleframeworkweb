    1.2

    • Added a main menu, where you can choose the other two scenes.

    • Added the Backup functionality, which allows the possibility of restarting a round when the player loses, restarting the characters and items as if nothing had happened. It is also possible to restart from the beginning of the current stage.

    • Added a menu when lost to restart the round, stage or return to the main menu.

    • Added a BattleState called Change Scene State, when it is reached the scene indicated by the user is loaded.

    • Updated documentation.
